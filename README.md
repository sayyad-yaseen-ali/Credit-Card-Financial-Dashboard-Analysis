# Credit-Card-Financial-Dashboard-Analysis
📊 Credit Card Financial Dashboard Analysis
📌 Project Overview

The Credit Card Financial Dashboard project focuses on analyzing credit card transaction and customer data to monitor weekly performance and financial trends.

Organizations generate large volumes of transaction data every week, making manual analysis inefficient and time-consuming. This project transforms raw SQL data into actionable insights using Power BI, enabling real-time monitoring and data-driven decision-making.

🎯 Project Objectives

Analyze credit card transaction and customer data

Monitor weekly revenue and transaction trends

Track Week-over-Week (WoW) performance changes

Reduce manual reporting effort

Improve visibility into customer segmentation

Enable data-driven financial decision-making

❗ Problem Statement

Before this solution:

Weekly reports were prepared manually

WoW changes were difficult to track

Limited visibility into revenue contribution by customer segment

SQL data was underutilized for analytics

A centralized and automated dashboard solution was required to generate quick, accurate, and actionable insights.

🛠 Tools & Technologies Used

Power BI – Data visualization & dashboard development

MySQL Database – Transaction and customer data storage

SQL Workbench – Data extraction and querying

🔄 Methodology

Extracted CSV data into MySQL using SQL queries

Imported structured data into Power BI

Cleaned data (removed duplicates & invalid records)

Created calculated columns:

Age Group

Income Group

Developed DAX Measures:

Revenue

Current Week Revenue

Previous Week Revenue

Week Number

Designed interactive visuals and KPI cards

Built an interactive financial dashboard
